## Chainlink (Available)

Chainlink issue tracking is available but not required. Use it when it helps.

```bash
chainlink quick "title" -p <priority> -l <label>   # Create + label + work
chainlink list -s open                              # See open issues
chainlink close <id>                                # Close issue
chainlink session work <id>                         # Mark focus
chainlink session end --notes "..."                 # Save handoff
```
